# 图片对比工具

一个为日常设计审稿、版本回归、截图核对而做的桌面图片对比工具。  
支持拖拽、粘贴、多标签页、滑动分割线、灰度切换、工程保存，适合快速对比「改前 / 改后」。

> 当前仓库为中文版说明，演示截图与 GIF 后续可继续补充。

## 项目亮点

- 拖入两张图即可开始对比，打开速度直接，交互成本低
- 支持滑动分割线对比，更适合检查局部改动
- 支持多标签页并行查看多个方案或版本
- 支持粘贴图片，适合直接处理剪贴板截图
- 支持黑白 / 彩色切换，便于观察结构和明暗变化
- 支持保存为工程文件，保留两张图与标题样式配置
- 支持窗口置顶，方便和设计稿、浏览器、开发工具并排工作

## 适合什么场景

- UI 改版前后对比
- 网页截图回归检查
- 设计稿与实现稿核对
- Banner、海报、封面等视觉版本比较
- 修图前后效果确认
- 产品界面微调差异检查

## 快捷键

- `Ctrl + T`：新建标签页
- `Ctrl + W`：关闭当前标签页
- `Ctrl + C`：复制当前对比图
- `Ctrl + S`：保存当前工程
- `Tab`：交换 A / B 位置
- `Space`：切换黑白 / 彩色
- `1`：仅显示 A
- `2`：仅显示 B
- `F`：打开标题设置
- `T`：切换窗口置顶

## 鼠标操作

- 左键拖动：移动对比分割线
- 右键拖动：平移画布
- 右键单击：适配窗口
- 滚轮：缩放画面
- 中键单击：恢复原比例

## 运行方式

### 方式一：直接运行入口脚本

```bash
python image_compare_tool.py
```

### 方式二：运行拆分后的模块入口

```bash
python -m image_compare_tool.app
```

## 环境依赖

- Python 3.10+
- PySide6
- Pillow

安装依赖示例：

```bash
pip install PySide6 Pillow
```
